# Sales Analysis Report | Power BI Dashboard

## Project Overview
This Power BI dashboard provides an interactive sales analysis report designed to track business performance across key sales metrics and dimensions. The report helps users analyze **Units Sold, Gross Sales, Discounts, Net Sales, Cost of Goods, and Profit** across **Period, Segment, Country, and Product**. It is built to support quick performance monitoring, trend analysis, and business decision-making through an intuitive and modern dashboard design.

## Objectives
- Track core sales and profitability KPIs in one place
- Analyze performance across multiple business dimensions
- Enable users to explore trends by period, segment, country, and product
- Provide an interactive reporting experience using navigation, tooltips, and filters
- Support data-driven decisions through clear visual storytelling

## Tech Stack
- **Power BI Desktop**
- **Power Query**
- **DAX**
- **Interactive dashboard design**
- **Data modeling and visualization**

## Dashboard Landing Page
The dashboard opens on the **Home** page, which serves as the introduction screen for the report. This page gives users an overview of the project and provides quick summary cards for major KPIs such as **Units Sold, Net Sales, Cost of Goods, and Net Profit**. From this page, users can navigate to the detailed analysis pages using the tabs and menu options.

## Key KPIs
- **Units Sold** – total quantity of products sold
- **Gross Sales** – total revenue before discounts
- **Discounts** – total discount amount applied
- **Net Sales** – revenue after discounts
- **Cost of Goods** – total product cost
- **Profit** – final earnings after deducting costs

## Dashboard Features
- Interactive KPI cards for quick performance tracking
- Navigation from the **Home** page to detailed analysis pages
- Analysis by **Period, Segment, Country, and Product**
- Custom tooltip pages for deeper insights
- Export button to download the full dataset
- Clean and modern user-friendly dashboard design
- Summary table with KPI breakdown and profit margin

## Navigation Flow
The dashboard is designed with page-based navigation for a better user experience:
- The report first lands on the **Home** page
- Clicking on the tabs beside **Home** takes the user to the respective analysis pages such as:
  - **Sales Analysis**
  - **Units Sold**
  - **Net Sales**
  - **Net Profit**
  - **Cost of Goods**
  - Tooltip pages for detailed metric insights

This navigation structure makes it easier for users to move between overview and detailed analysis sections.

## Key Analysis Areas
### 1. Period Analysis
Users can analyze how sales and profitability metrics change across different time periods.

### 2. Segment Analysis
The dashboard compares KPI performance across different customer or business segments.

### 3. Country Analysis
Users can evaluate sales contribution and profitability by country.

### 4. Product Analysis
The report helps identify top-performing and low-performing products based on sales and profit metrics.

### 5. Profitability Analysis
By comparing discounts, net sales, and cost of goods, users can understand the factors affecting profit and profit margin.

## Export Functionality
An **Export** button is included in the dashboard, allowing users to extract the full underlying dataset for further analysis and reporting.

## Business Value
This dashboard makes it easier to monitor business performance, identify high-performing areas, track profitability trends, and export data for further analysis. It provides a complete and interactive view of sales operations in a visually appealing format.
