# Bicycle-Sales-Analysis

> An interactive Bicycle sales analytics dashboard by the employes during the Sep 2021 to Dec 2022 — exploring Units Sold, Net Sales, Profit and Cost of Goods measured by Period, Segment, Country and Product.

---
## 🎯 Objectives
- Analyze Bicycle sales/performance/customer/product trends.
- **Visualize data** using interactive Power BI dashboards.
- Enable filtering, drill-through, and tooltip analysis.
- **Derive insights** to improve business decision-making.

## 🛠️ Tech Stack
- **Power BI** – Dashboard creation & visualization.
- **Excel** – Data preprocessing.
